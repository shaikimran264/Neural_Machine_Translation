# ──────────────────────────────────────────────
# POST-INSTALL STEPS  (run these after pip install)
# ──────────────────────────────────────────────
#
# 1. Download the spaCy English model:
#       python -m spacy download en_core_web_sm
#
# 2. Download Hindi FastText binary vectors (~4 GB):
#       wget https://dl.fbaipublicfiles.com/fasttext/vectors-crawl/cc.hi.300.bin.gz
#       gunzip cc.hi.300.bin.gz
#
# 3. Download GloVe 300-d English vectors (~1 GB):
#       wget https://nlp.stanford.edu/data/glove.6B.zip
#       unzip glove.6B.zip   # use glove.6B.300d.txt
#
# 4. Place your dataset JSON at the path expected by the notebook:
#       2_merged_train_only (1).json   – training split
#       (test JSON loaded from Google Drive / local path)
#
# ──────────────────────────────────────────────
# NOTE ON fasttext vs fasttext-wheel
# ──────────────────────────────────────────────
# The notebook installs `fasttext` via `pip install fasttext`.
# On some systems (especially Linux/Colab) the plain `fasttext`
# package compiles from source and may fail without build tools.
# `fasttext-wheel` provides a pre-built wheel and is a drop-in
# replacement.  If you are on Colab, the plain `fasttext` package
# works fine – you can replace fasttext-wheel with fasttext.
